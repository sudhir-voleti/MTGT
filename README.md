# MTGT — Marketing Tools for Go-to-Market

**ISB PGP YL Cohort 2025-27**  
**Course by Prof. Sudhir Voleti**

This repository contains materials for the **Marketing Tools for Go-to-Market (MTGT)** elective course at the Indian School of Business.

## Repository Structure

- `Lec01/` — Lecture 1 materials (Caselet 1: DevSuite India, Gradio apps, data generators)
- `Lec02/` — Lecture 2 materials (Caselet 2: FinScale, competitive displacement)
- `apps/` — Interactive Gradio apps for in-class exercises
- `data/` — Synthetic datasets for caselets
- `prompt_guides/` — Prompt guides for NotebookLM / LLM analysis

## How to Use

1. Open any Gradio app notebook in Google Colab.
2. Run the first cell to install dependencies.
3. Upload the relevant CSV file (e.g., `devsuite_workspace_logs.csv`).
4. Follow the tabs: Data Preview → Segmentation → Traction Mapping → Results.

## Course Focus

- First-principles GTM strategy using the **Traction Equation**
- Customer segmentation and traction analysis
- Competitive intelligence and displacement modeling
- Hands-on use of AI tools (Gradio, NotebookLM, Colab)
